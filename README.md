# 🚉 Station Management System (C Project)

## 📌 Description
This project is a station management system written in C using a doubly linked list.  
It allows full management of stations including insertion, deletion, search, update, sorting, and file handling.

## ⚙️ Features

- Add station (beginning / end / after ID)
- Delete station (by ID / first / last)
- Search station by ID
- Display stations (forward and reverse)
- Filter by quartier
- Update station information
- Statistics (total stations, average travelers, min/max)
- Sorting by number of travelers
- Auto-generate random stations
- Save / load data from file
- Logging system (tracks all operations)

---

## 🧠 Concepts Used

- Doubly linked list
- Dynamic memory allocation (malloc/free)
- File handling
- String manipulation
- Modular programming (multiple files)
- Logging system
